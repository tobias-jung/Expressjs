Grundlagen

Das Express Framework (Express.js) ist ein serverseitiges Web Applikation Framework, welches Werkzeuge zur Entwicklung von Webanwendungen bereitstellt.

Express.js enthält ein durchgängig umgesetztes Middleware-Konzept, um die entkoppelte Entwicklung von simplen Webservern bis zu ausgefeilten Webservices zu ermöglichen.

Um Express.js nutzen zu können, muss Node.js und Express.js auf dem Server installiert werden!

Anwendungsbeispiele

Projektanwendungsbeispiele für Express.js bzw. der Implementierung von Node.js:


Installation Express.js

1. Erstelle das JASON-Package für die Applikation:


                                npm init

2. Geben Sie den Namen der Datei an:


                                entry point: (app.js)

3. Im Arbeitsverzeichnis Express.js installieren:


                                npm install express

Die app.js

Nachfolgender Code steht in der app.js. Er startet einen Webserver und hört auf den Port: 3000:

var app = express();
var server = app.listen(3001, function () {
var host = server.address().address;

console.log('Example app listening at http://%s:%s', host, port);

GET-Methode

So kann eine einfache GET-Methode aussehen:


app.get('/', function (req, res) {
res.send('Hello World!');
});

Es wird beim Aufruf "Hello World!" zurückgegeben!

POST-Methode

So kann eine einfache POST-Methode aussehen:


app.post('/', function (req, res) {
res.send('Ich habe einen Post erhalten!');

Fazit

Mithilfe von Express.js kann serverseitig eine Web-Applikation zur Verfügung gestellt werden!

Voraussetzung für die Verwendung ist die Installation von Node.js.

Als Kommunikationsschnittstelle für den Webserver dient in unserem Beispiel die Applikation app!

Bekannte Funktionen sind app.get() und app.post()!
